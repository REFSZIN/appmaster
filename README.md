# Description

## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd projet
```

Install dependencies

```bash
  npm install
```


### Compiles and minifies for production
```bash
  npm run build
```

### Compiles and hot-reloads for development
```bash
  npm run start
```


Run DOCKER  

```bash
  docker compose up --build -d
```
